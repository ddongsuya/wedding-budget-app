## 🔧 Sentry 설정 (선택사항)

현재 Sentry는 **선택적으로 비활성화** 상태입니다.
DSN이 없어도 앱이 정상 작동합니다.

### Sentry 활성화 방법

#### 1단계: Sentry 계정 생성
1. https://sentry.io 접속
2. 무료 계정 생성 (GitHub 로그인 가능)

#### 2단계: 프로젝트 생성

**프론트엔드 프로젝트**
1. "Create Project" 클릭
2. Platform: **React** 선택
3. Project name: `wedding-budget-frontend`
4. DSN 복사

**백엔드 프로젝트**
1. "Create Project" 클릭
2. Platform: **Node.js** 선택
3. Project name: `wedding-budget-backend`
4. DSN 복사

#### 3단계: 환경 변수 추가

**Vercel (프론트엔드)**
1. Vercel 대시보드 → 프로젝트 선택
2. Settings → Environment Variables
3. 추가:
   - Name: `VITE_SENTRY_DSN`
   - Value: `프론트엔드 DSN`
   - Name: `VITE_APP_VERSION`
   - Value: `1.0.0`
4. "Save" 클릭
5. Deployments → 최신 배포 → "Redeploy"

**Render (백엔드)**
1. Render 대시보드 → 서비스 선택
2. Environment → Environment Variables
3. 추가:
   - Key: `SENTRY_DSN`
   - Value: `백엔드 DSN`
   - Key: `APP_VERSION`
   - Value: `1.0.0`
4. "Save Changes" 클릭
5. 자동 재배포 시작

#### 4단계: 테스트
1. 의도적으로 에러 발생
2. Sentry 대시보드에서 확인

## 📱 앱 사용 방법

### 1. 회원가입/로그인
1. 앱 접속
2. "회원가입" 클릭
3. 이메일, 비밀번호, 이름 입력
4. 로그인

### 2. 프로필 설정
1. 설정 메뉴 이동
2. 신랑/신부 정보 입력
3. 결혼 날짜 설정
4. 저장

### 3. 예산 설정
1. 예산 관리 메뉴
2. 총 예산 입력
3. 카테고리별 예산 배분
4. 저장

### 4. 지출 기록
1. 지출 내역 메뉴
2. "지출 추가" 클릭
3. 항목명, 금액, 날짜 등 입력
4. 저장

### 5. 식장 비교
1. 웨딩홀 비교 메뉴
2. "식장 추가" 클릭
3. 식장 정보 입력
4. 여러 식장 등록 후 비교

### 일반적인 문제

**로그인이 안 돼요**
- 이메일/비밀번호 확인
- 백엔드 서버 상태 확인
- 네트워크 연결 확인

**데이터가 안 보여요**
- 로그인 상태 확인
- API 응답 확인 (개발자 도구)
- 백엔드 로그 확인

**PWA 설치가 안 돼요**
- HTTPS 연결 확인
- 브라우저 지원 확인 (Chrome, Edge, Safari)
- manifest.json 확인
