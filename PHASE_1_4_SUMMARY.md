## 📦 구현 내용

### 1. 생성된 컴포넌트 (5개)

#### EmptyState.tsx
- **역할**: 기본 빈 상태 컴포넌트
- **기능**: 
  - 일러스트레이션 또는 아이콘 표시
  - 타이틀, 설명 표시
  - 주요/보조 액션 버튼
  - 애니메이션 효과
- **크기**: ~2KB

#### EmptyStateIllustration.tsx
- **역할**: SVG 일러스트레이션 제공
- **종류**: 6가지 (wedding, budget, expense, venue, checklist, calendar)
- **크기**: ~8KB (전체)

#### NoSearchResults.tsx
- **역할**: 검색 결과 없음 상태
- **기능**: 검색어 표시, 검색어 지우기 버튼
- **크기**: ~1KB

#### ErrorState.tsx
- **역할**: 에러 상태 표시
- **기능**: 에러 메시지, 재시도 버튼
- **크기**: ~1KB

#### index.ts
- **역할**: Export 통합
- **기능**: 컴포넌트 일괄 export

**총 번들 크기 증가**: ~12KB

## 🎨 디자인 원칙

### 1. 색상 조화
- 앱의 베이지/로즈골드 톤과 조화
- 각 기능별 구분되는 색상 팔레트
- 부드럽고 따뜻한 느낌

### 2. 문구 톤
**✅ 좋은 예시**:
- "아직 기록된 지출이 없어요"
- "마음에 드는 웨딩홀을 등록하고 비교해보세요"
- "첫 지출 기록하기"

**❌ 피해야 할 예시**:
- "데이터가 없습니다"
- "지출을 추가하세요"
- "확인"

**원칙**:
- 부정적이지 않고 긍정적
- 친근하고 대화하는 느낌
- 명확한 행동 유도

### 3. 버튼 디자인
**Primary Action**:
- 그라데이션 로즈 컬러
- 호버 시 확대 효과
- 그림자로 입체감

**Secondary Action**:
- 중립적인 회색 톤
- 주요 액션보다 덜 강조
