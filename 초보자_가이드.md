## 1단계: 데이터베이스 만들기 (5분)

### 방법 1: 자동 스크립트 사용 (추천!)

1. **파일 탐색기 열기**
   - 키보드에서 `Windows 키 + E` 누르기
   - 또는 작업 표시줄에서 폴더 아이콘 클릭

2. **프로젝트 폴더로 이동**
   - 바탕화면 → wedding 폴더 열기
   - `backend` 폴더 열기

3. **create-db.bat 파일 찾기**
   - `create-db.bat` 파일이 보일 거예요
   - 이 파일을 **더블클릭** (두 번 빠르게 클릭)

4. **비밀번호 입력**
   - 검은 창(명령 프롬프트)이 열립니다
   - "Password for user postgres:" 라고 나옵니다
   - PostgreSQL 설치할 때 입력했던 비밀번호를 입력하세요
   - **주의**: 비밀번호를 입력해도 화면에 아무것도 안 보입니다 (정상입니다!)
   - 입력 후 `Enter` 키 누르기

5. **결과 확인**
   - "Database created successfully!" → 성공! ✅
   - "already exists" → 이미 만들어져 있음 (괜찮아요!) ✅
   - 아무 키나 눌러서 창 닫기

### 방법 2: 수동으로 만들기 (방법 1이 안 되면)

1. **시작 메뉴 열기**
   - 키보드에서 `Windows 키` 누르기
   - 또는 화면 왼쪽 아래 Windows 아이콘 클릭

2. **SQL Shell 찾기**
   - 검색창에 "SQL Shell" 입력
   - "SQL Shell (psql)" 프로그램이 보이면 클릭

3. **연결 정보 입력**
   - 여러 질문이 나옵니다
   - **모두 그냥 Enter만 누르세요** (5번)
   ```
   Server [localhost]: (Enter 누르기)
   Database [postgres]: (Enter 누르기)
   Port [5432]: (Enter 누르기)
   Username [postgres]: (Enter 누르기)
   Password for user postgres: (비밀번호 입력 후 Enter)
   ```

4. **데이터베이스 생성**
   - `postgres=#` 이렇게 나오면 성공!
   - 다음 명령어를 **정확히** 입력하세요:
   ```sql
   CREATE DATABASE wedding_budget;
   ```
   - `Enter` 키 누르기
   - "CREATE DATABASE" 라고 나오면 성공! ✅

5. **종료**
   - `\q` 입력 후 `Enter`
   - 창이 닫힙니다

## 2단계: 데이터베이스 테이블 만들기 (3분)

### VS Code 터미널 사용하기

1. **VS Code 열기**
   - 프로젝트 폴더가 열려있는 VS Code 창으로 가세요
   - 없으면: 바탕화면 → wedding 폴더 우클릭 → "Code로 열기"

2. **터미널 열기**
   - 상단 메뉴에서 `터미널` → `새 터미널` 클릭
   - 또는 키보드 단축키: `Ctrl + ` (백틱, 숫자 1 왼쪽 키)
   - 화면 아래에 터미널 창이 나타납니다

3. **backend 폴더로 이동**
   - 터미널에 다음 명령어 입력:
   ```bash
   cd backend
   ```
   - `Enter` 키 누르기
   - 터미널 경로가 `...wedding\backend>` 로 바뀝니다

4. **마이그레이션 실행**
   - 다음 명령어 입력:
   ```bash
   npm run migrate
   ```
   - `Enter` 키 누르기
   - 잠시 기다리면...

5. **결과 확인**
   - "Migrations completed successfully!" 메시지가 나오면 성공! ✅
   - 에러가 나오면 에러 메시지를 복사해서 알려주세요

## 3단계: 서버 실행하기 (2분)

이제 백엔드와 프론트엔드 서버를 실행해야 합니다.

### 방법 1: 배치 파일 사용 (가장 쉬움!)

1. **파일 탐색기 열기**
   - `Windows 키 + E`

2. **프로젝트 폴더로 이동**
   - 바탕화면 → wedding 폴더

3. **백엔드 서버 시작**
   - `start-backend.bat` 파일 **더블클릭**
   - 검은 창이 열리고 코드가 주르륵 나옵니다
   - "Server running on port 3000" 메시지가 나오면 성공! ✅
   - **이 창을 닫지 마세요!**

4. **프론트엔드 서버 시작**
   - 다시 wedding 폴더로 가서
   - `start-frontend.bat` 파일 **더블클릭**
   - 또 다른 검은 창이 열립니다
   - "Local: http://localhost:5173" 메시지가 나오면 성공! ✅
   - **이 창도 닫지 마세요!**

### 방법 2: VS Code 터미널 사용

1. **첫 번째 터미널 (백엔드)**
   - VS Code에서 터미널 열기 (`Ctrl + ` )
   - 명령어 입력:
   ```bash
   cd backend
   npm run dev
   ```
   - "Server running on port 3000" 나오면 성공! ✅

2. **두 번째 터미널 (프론트엔드)**
   - 터미널 오른쪽 위에 `+` 버튼 클릭 (새 터미널)
   - 명령어 입력:
   ```bash
   npm run dev
   ```
   - "Local: http://localhost:5173" 나오면 성공! ✅

## ❓ 문제 해결

### "비밀번호가 틀렸습니다"
- PostgreSQL 설치할 때 설정한 비밀번호를 다시 확인하세요
- 기억이 안 나면 PostgreSQL을 재설치하세요

### "포트가 이미 사용 중입니다"
- 다른 프로그램이 같은 포트를 사용하고 있습니다
- 열려있는 터미널이나 명령 프롬프트 창을 모두 닫고 다시 시도하세요

### "npm을 찾을 수 없습니다"
- Node.js가 설치되지 않았습니다
- https://nodejs.org 에서 다운로드 후 설치하세요

### "데이터베이스에 연결할 수 없습니다"
1. PostgreSQL이 실행 중인지 확인:
   - `Windows 키` → "서비스" 검색
   - "postgresql" 찾아서 "실행 중" 상태 확인
2. 비밀번호가 맞는지 확인
3. `backend/.env` 파일에서 `DB_PASSWORD` 확인

### 그래도 안 되면?
- 에러 메시지를 **정확히** 복사해서 알려주세요
- 어느 단계에서 막혔는지 알려주세요
- 스크린샷을 찍어서 보여주세요
